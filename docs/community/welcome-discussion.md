# Welcome discussion (paste when enabling GitHub Discussions)

After enabling Discussions on the repo (**Settings → General → Features → Discussions**), create a pinned **General** post with this body:

---

**Welcome to HeatLens**

HeatLens shows how much heat your PC is putting into the room — live watts, BTU/hr (or kW), session heat, temps, and rough CFM / still-air estimates when you enter ambient temperature.

**Quick links**
- [Releases](https://github.com/arogorn993-hue/HeatLens/releases) — portable Windows / Linux / macOS builds
- [README](https://github.com/arogorn993-hue/HeatLens#quick-start) — install and sensor tips

**Good topics for this board**
- Show and tell — your setup and screenshots
- Q&A — sensors, accuracy, LibreHardwareMonitor, Low impact mode
- Ideas — features you'd like to see

**Accuracy note:** Software power readings are useful for context; a wall meter is still the gold standard for whole-system draw.

Issues are for bugs and concrete feature requests. Everything else fits here.
